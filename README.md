# BBR Weather

Flask-based weather application built for DeerMove / Big Buck Registry.

## Features
- Current conditions
- 10-day forecast
- Expandable 10-day hourly breakdown
- Moon phase with illumination
- Wind direction + pressure trend

## Tech Stack
- Python / Flask
- Visual Crossing Weather API
- OpenCage Geocoding
- Render hosting

## Notes
- Render Starter plan limits custom domains per workspace
- Currently deployed on onrender.com
- Custom domain pending future consolidation

## Run Locally
```bash
pip install -r requirements.txt
python app.py
